About

AA Studio is a minimal, standalone web app for crafting blocky or UTF‑8 art in your browser. No installation, no backend, no build tools—just open index.html and start creating.

  • Draw with any Unicode symbols in a textarea
  • Export/import your art as JSON files
  • Change fonts, font size, and color themes
  • Customizable symbol palette
  • Mobile and desktop friendly

Built with vanilla JavaScript and styled using Tailwind CSS via CDN.
